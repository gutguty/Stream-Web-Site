# Stream Web Site - Стриминговый сервис

> Frontend приложение для стримингового сервиса с компонентной архитектурой

## Функциональность

- Компонентная архитектура с использованием JSX
- Cборка проекта через Vite
- Оптимизация итогового бандла
- Адаптивный пользовательский интерфейс
- Навигация между разделами стриминга

## Технологии

HTML, React, JavaScript, JSX, CSS3, Vite, Swiper
  
## Установка зависимостей

```bash
git clone https://github.com/gutguty/Stream-Web-Site.git

npm install

npm run start

npm run build

npm run preview
```
